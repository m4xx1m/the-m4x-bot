## Setup
* Clone the repo
* Optional
  * Generate venv
    ```sh
    $ python3 -m venv venv
    ```
  *  Activate venv
    ```sh
    $ source venv/bin/activate
    ```
* Setup requirements
    ```sh
    $ python3 -m pip install -r requirements.txt
  ```

* Run bot
  ```sh
    $ python3 -m bot
  ```
